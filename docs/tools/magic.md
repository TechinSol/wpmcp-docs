# Magic Tools

Magic Tools are high-level tools that chain multiple actions together in one command. They're designed for non-technical users who want to get a lot done with a single prompt.

---

## wp_magic_site_setup
**Level:** 🟡 WRITE

Set up your entire site in one go — name, tagline, colors, and key pages.

```
Set up my WordPress site:
- Site name: "Bright Blooms Florist"
- Tagline: "Fresh flowers, happy moments"
- Primary color: #16a34a
- Create pages: Home, About, Services, Contact
```

---

## wp_magic_launch_blog
**Level:** 🟡 WRITE

Write multiple blog posts on a topic and publish them all at once.

```
Write 3 blog posts about "indoor plant care for beginners"
and publish them as drafts so I can review them first.
```

---

## wp_magic_seo_sweep
**Level:** 🔵 READ + 🟡 WRITE

Audit all posts and pages for SEO issues, then optionally fix them.

```
Run a full SEO sweep on my site — check every post and page
for missing titles, meta descriptions, and thin content.
Give me a prioritized list of what to fix.
```

---

## wp_magic_bulk_add_products
**Level:** 🟡 WRITE

Add multiple WooCommerce products in a single request.

```
Add these products to my WooCommerce store:
1. Red Mug — $12.99 — 50 in stock
2. Blue Notebook — $7.49 — 100 in stock
3. Wooden Pen — $4.99 — 200 in stock
```

---

## wp_magic_color_palette
**Level:** 🟡 WRITE

Generate a harmonious color palette from a single seed color and apply it to your theme.

```
Generate a professional color palette for a wellness brand
starting from #0d9488 and apply it to my WordPress theme.
```

---

## wp_magic_duplicate_post
**Level:** 🟡 WRITE

Copy any page or post as a new draft.

```
Duplicate my "Services" page and save the copy
as a draft called "Services — New Version".
```

---

## wp_magic_create_landing_page
**Level:** 🟡 WRITE

Create a complete landing page with headline, benefits, testimonial, and CTA.

```
Create a landing page for my online course
"Master Watercolor Painting" — include a headline,
5 benefits, a testimonial, and a call-to-action.
Save as draft.
```

---

## wp_magic_content_audit
**Level:** 🔵 READ

Audit all your content for quality issues — short posts, missing images, no categories.

```
Audit all my blog posts — flag any that are under 300 words,
missing a featured image, or have no category assigned.
```
