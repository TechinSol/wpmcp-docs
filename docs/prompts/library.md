# Prompt Library

Ready-to-use prompts for every WP-MCP tool. Copy any prompt and paste it directly into Claude or Codex.

:::tip How to use
Copy a prompt → open Claude or Codex → paste it in → the right tool runs automatically.
:::

---

## ✨ Magic Tools

These prompts trigger high-level tasks that chain multiple tools together.

**Full site setup**
```
Set up my WordPress site: name "Bright Blooms Florist",
tagline "Fresh flowers, happy moments", primary color #16a34a.
Create pages: Home, About, Services, Contact.
```

**Launch a blog**
```
Write 3 blog posts about "indoor plant care for beginners"
and publish them as drafts so I can review before going live.
```

**Full SEO audit**
```
Run a full SEO audit on my WordPress site — check every post and page
for missing titles, meta descriptions, thin content, and images without alt text.
Give me a prioritized list.
```

**Bulk add products**
```
Add these products to my WooCommerce store:
1. Red Mug — $12.99 — 50 in stock
2. Blue Notebook — $7.49 — 100 in stock
3. Wooden Pen — $4.99 — 200 in stock
```

## 📝 Content

**Write a blog post**
```
Write a blog post about "10 tips for small business owners"
with at least 600 words. Add a category "Business Tips" and save as draft.
```

**Publish a draft**
```
Publish my draft post "10 tips for small business owners".
```

**Duplicate a page**
```
Duplicate my "Services" page and save the copy
as a draft called "Services — New Version".
```

**Create a Contact page**
```
Create a Contact page for my site. Include our office address
(123 Main St, Austin TX), phone (555-0100), and email (hello@mysite.com). Publish it.
```

**Find all drafts**
```
List all draft posts and pages on my site — show title, last modified date, and author.
```

## 🔍 SEO

**Audit one page**
```
Run an SEO audit on my "Home" page and tell me exactly what to improve.
```

**Set Yoast meta**
```
Set the Yoast SEO meta for my "About Us" page:
- Title: "About [Your Business] — [Your City]"
- Description: "We [what you do]. [Value prop]. [CTA]."
- Focus keyphrase: "[your main keyword]"
```

**Full site sweep**
```
Check all my posts for missing SEO titles, meta descriptions,
and featured images. Give me a prioritized fix list.
```

## 🎨 Design

**Change colors**
```
Update my site's primary color to navy blue (#1e3a8a),
accent color to gold (#f59e0b), and tagline to "We help small businesses grow".
```

**Generate a palette**
```
Generate a professional color palette for a [industry] brand
starting from the color [hex code] and apply it to my theme.
```

## 🛒 WooCommerce

**List products**
```
Show me all products in my store — name, price, stock status, and category.
```

**Update a price**
```
Update the price of "[Product Name]" to $[price]
and set the stock quantity to [number].
```

**Recent orders**
```
Show me my last 10 orders — order number, customer name, total, and status.
```

## 🏥 Health

**Full health check**
```
Run a complete health check on my WordPress site — check PHP version,
active plugins, SSL, debug mode, and give me an overall score.
```

**Check error log**
```
Show me the last 50 lines of my WordPress debug.log
so I can check for PHP errors.
```

**Clean spam**
```
How many spam comments do I have? Then trash all of them.
```
