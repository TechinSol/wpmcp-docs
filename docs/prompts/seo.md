# SEO Prompts

Prompts for auditing, fixing, and optimizing SEO across your WordPress site.

---

## Site-wide audits

**Full SEO sweep**
```
Run a full SEO audit on my site — check every post and page
for missing titles, meta descriptions, and thin content.
Give me a prioritized list of what to fix, starting with the most important.
```

**Find pages missing meta**
```
List every post and page that is missing an SEO title or meta description.
Group them by post type.
```

**Find short posts**
```
List all published blog posts that are under 400 words.
These may need to be expanded for better SEO.
```

---

## Single page audits

**Detailed page audit**
```
Run an SEO audit on my "[page title]" page and tell me
exactly what to improve — title, description, keyword usage,
content length, and images.
```

---

## Setting meta tags

**Set Yoast meta**
```
Set the Yoast SEO meta for my "[page title]" page:
- SEO title: "[title]"
- Meta description: "[description — 150-160 characters]"
- Focus keyphrase: "[keyword]"
```

**Set Rank Math meta**
```
Set the Rank Math SEO meta for my "[page title]" page:
- SEO title: "[title]"
- Focus keyword: "[keyword]"
- Meta description: "[description]"
```

**Bulk update meta**
```
For each of my top 5 blog posts, write an optimized SEO title
and meta description targeting the post's main keyword.
Then apply them.
```

---

## Image SEO

**Fix missing alt text**
```
Find all images in my media library that have no alt text.
List them so I can update them one by one.
```

**Set alt text**
```
Update the alt text for the image "[filename]" to
"[descriptive alt text]".
```
