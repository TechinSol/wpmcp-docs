# Design Prompts

Prompts for changing your site's appearance using plain English.

---

## Colors & branding

**Change primary color**
```
Update my WordPress site's primary color to [color name or hex code].
```

**Full color update**
```
Update my theme colors:
- Primary: [hex] ([color name])
- Secondary: [hex] ([color name])
- Background: [hex]
- Text: [hex]
```

**Generate a palette**
```
Generate a professional color palette for a [industry] brand
starting from the color [hex code].
Apply it to my WordPress theme automatically.
```

**Update site name and tagline**
```
Update my site name to "[name]" and tagline to "[tagline]".
```

---

## Fonts & typography

**Change font**
```
Update my theme's body font to [font name]
and heading font to [font name].
```

---

## Theme management

**Check current theme**
```
What theme is currently active on my site?
List all other installed themes too.
```

**Switch theme**
```
Switch my active WordPress theme to "[theme name]".
```

:::warning
Switching themes changes your entire site's appearance immediately. Claude will ask you to confirm.
:::

---

## Page layout (Gutenberg)

**Update a heading**
```
Update the main heading on my "[page name]" page to
"[new heading text]".
```

**Add a section**
```
Add a new section at the bottom of my "[page name]" page
with the heading "[heading]" and this paragraph:
"[content]"
```

---

## Page layout (Elementor)

**Read layout**
```
Show me the structure of my "[page name]" Elementor page —
sections, columns, and widget types.
```
