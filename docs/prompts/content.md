# Content Prompts

Prompts for writing, editing, and managing WordPress posts and pages.

---

## Writing posts

**Blog post (draft)**
```
Write a blog post about "[topic]" with at least 600 words.
Add it to the category "[category]" and save as a draft.
```

**Blog post with SEO**
```
Write a blog post about "[topic]". Target the keyword "[keyword]".
Include a meta title and meta description. Save as draft.
```

**Multiple posts at once**
```
Write 3 blog posts about "[topic]" — one per post.
Save them all as drafts so I can review before publishing.
```

---

## Creating pages

**About page**
```
Create an About page for my business.
We are a [type of business] based in [city] that [what you do].
Founded in [year]. Publish it.
```

**Services page**
```
Create a Services page listing these services:
1. [Service 1] — [price or description]
2. [Service 2] — [price or description]
3. [Service 3] — [price or description]
Publish it.
```

**Contact page**
```
Create a Contact page with:
- Address: [your address]
- Phone: [your phone]
- Email: [your email]
- Hours: [your hours]
Publish it.
```

**Landing page**
```
Create a landing page for [product/offer].
Include: a headline, 5 bullet-point benefits, one testimonial,
and a call-to-action button. Save as draft.
```

---

## Editing content

**Update a page**
```
Update my "[page name]" page — change the headline to
"[new headline]" and add this paragraph at the end:
"[new paragraph]"
```

**Publish a draft**
```
Publish my draft post "[post title]".
```

**Bulk-publish drafts**
```
List all my draft posts, then publish every one that was created
more than 30 days ago.
```

---

## Organizing content

**Create a category**
```
Create a new blog category called "[name]" with the slug "[slug]".
```

**Reassign posts**
```
Move all posts in the category "Uncategorized" to the category "Blog".
```

**Find posts missing a category**
```
Find all posts that have no category assigned.
```
